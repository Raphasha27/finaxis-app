## 📱 Mobile App (React Native + Expo Go)
The mobile experience has been officially transitioned to **Expo Go** for rapid development and high-fidelity native interactions.

### Key Mobile Features:
- **Dual Dashboard Hub**: Seamlessly switch between **Attendee** (spending) and **Organizer** (analytics) roles.
- **Event Wallet (Settlement Core)**: Integrated credit management with atomic ledger updates.
- **AI Event Butler**: Interactive assistant for lodging, directions, and schedule management.
- **Vendor Analytics**: Real-time performance bars and revenue tracking for organizers.
- **Micro-Animations**: Native sheet modals and high-performance transitions.

### How to Run:
1. Ensure you have the [Expo Go](https://expo.dev/go) app on your mobile device.
2. Install dependencies:
   ```bash
   cd finaxis-expo
   npm install
   ```
3. Start the development server:
   ```bash
   npx expo start
   ```
4. Scan the QR code with your phone to launch the **FinAxis Premium Ecosystem**.
